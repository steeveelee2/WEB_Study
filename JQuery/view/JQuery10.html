객체 삽입 관련 메소드(객체 생성 및 제거)

삽입 메소드 part1

$(A).appendTo(B) : A를 B 요소 마지막 부분에 추가
$(A).prependTo(B) : A를 B 요소 앞부분에 추가
$(A).insertAfter(B) : A를 B 요소 뒤에 추가
$(A).insertBefore(B) : A를 B 요소 앞에 추가

삽입 메소드 part 2

$(A).append(B) : B를 A요소 안의 마지막 부분에 추가
$(A).prepend(B) : B를 A요소 안의 앞부분에 추가
$(A).after(B) : B를 A요소 뒤에 추가
$(A).before(B) : B를 A요소 앞에 추가

객체의 복사와 제거 메소드

clone() : 지정한 객체(요소)를 복사할 때 사용
remove() / detach() / empty() : 객체(요소)를 제거하거나 내용을 비울 때 사용
