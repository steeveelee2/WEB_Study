속성 선택자

속성 선택자는 기본 선택자 뒤에 정의하여 사용한다.

요소[속성] : 해당하는 속성을 가진 요소를 선택하는 방법
요소[속성명=속성값] : 해당 속성의 특정 값을 가진 요소
Ex) div[name='Hello'] : div 중 name이 Hello인 요소를 선택
    요소[속성명 ~= 속성값] : 해당 속성의 특정 값을 단어로써 포함하는 요소를 선택
    요소[속성명 $= 속성값] : 속성 안의 값이 '속성값'으로 끝나는 요소
    요소[속성명 *= 속성값] : 속성의 값 중 특정값이 존재하는 요소를 선택
Ex) <div name='hello1 hello2 hello3'> ----> div[name *= hello2];
