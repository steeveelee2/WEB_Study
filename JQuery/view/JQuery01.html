JQuery 개요

JQuery?

존 레식에 의해 개발된 JS 경량화 라이브러리로
복잡한 JS 구문을 짧고 쉽게 개발할 수 있도록 구현된 라이브러리이다.

-- JQuery 장점 --
1. DOM과 관련된 스크립트를 쉽게 구현할 수 있다.
2. ajax(비동기)통신, 이벤트 처리 등을 폭넓게 지원한다.
3. JQuery와 관련된 확장형 플러그인, 오픈 API를 다양하게 지원하며
 이를 통해 차트, 슬라이드, 애니매니션을 간단하게 구현할 수 있다.

JQuery 연결방법

1. JQuery 웹사이트에서 JS파일을 다운로드 한 후 연결하는 방법
2. CDN(Content Delivery Network)을 통해 연결하는 방법

$(documnet).ready()

현재 웹 문서를 로드한 후 ready 함수를 실행한다.
해당 내용은 documnet.onload와 동일하다.
