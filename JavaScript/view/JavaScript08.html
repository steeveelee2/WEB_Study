배열

자바 스크립트 배열 :

자바에서는 배열을 선언할 때 특정 자료형을 선언하여 해당 자료형의 값들만 받을 수 있지만
자바 스크립트에서는 자료형을 별도로 지정하지 않기 때문에 모든 자료형을 하나의 배열에 넣을 수 있다

배열 선언

배열 선언 시 new Array()를 사용하여 선언할 수도 있고 []를 사용하여 선언할 수도 있다
또한 배열 선언 시 크기를 지정하거나 지정하지 않아도 된다

array 객체의 메소드(함수)

array도 하나의 객체이다. 때문에 배열에서도 여러 종류의 함수를 사용할 수 있다.

indexOf() : 배열의 특정 요소가 몇번째인지 확인하여 해당 인덱스를 반환하는 함수

concat() : 두 개의 배열을 합치는 함수

join() : 배열을 하나의 문자열로 합치는 함수

reverse() : 배열의 순서를 반대로 뒤집는 함수

sor() : 배열을 정렬하는 함수

push() : 배열의 맨 뒤에 값을 추가하는 함수
pop() : 배열의 맨 뒤의 값을 꺼내어 제거하는 함수

slice() : 배열의 요소를 선택하여 잘라내는 함수
splice() : 배열의 인덱스 위치의 요소를 추가/제거하는 함수

toString() : 배열을 문자열로 변환하는 함수
