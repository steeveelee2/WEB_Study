이미지 관련 태그

자주 사용하는 속성들

alt 속성

화면에 이미지를 표시하거나, 이미지가 표시되지 않을 때 해당 이미지에 대한 간단한 설명을 달아주는 태그 웹 브라우저에서 화면 낭독기(스크린 리더)를 통해 화면의 내용을 읽어올 때 사용된다 화면 낭독기는 시각 장애인을 위한 웹 접근성 프로그램으로 화면의 내용을 보지 못하는 사용자에게 해당 화면의 내용들을 낭독하여 읽어준다

width와 height 속성

사진의 높이와 길이를 지정하는 속성들 사진의 크기는 고정된 크기와 가변 크기로 설정할 수 있다

고정 크기로 설정할 경우

화면(웹 브라우저)의 크기가 변해도 이미지의 크기는 변함없다

가변 크기로 설정할 경우

현재 웹 브라우저의 크기에 따라 이미지의 크기도 변경된다

map

이미지에 구역을 설정하여 각 구역에 링크나 특정 작업을 연결하는 태그
